Testcases
---------

The following files contain the text, output, and expected output for four testcases. The self testcase is a copy of the tokenizer.c source file. Testcase1 contains text to explicitly test for all possible token types and most (if not all) points of failure for tokens. Testcase2 and Testcase3 are source files for the linux kernel (time.c and cpu.c, respectively).
